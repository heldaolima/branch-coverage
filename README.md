# Branch Covering in Go

This project is a simple showcase of branch covering in the Go Programming Language. The main focus are the test files, which attempt to test all branches in the Merge Sort, Quick Sort and Binary Search code.

## How to Run
The `Makefile` contains the required commands. For running the branch coverage test, you should install the [gobco](https://github.com/rillig/gobco) package.

The command to run the branch coverage test is the following:

```shell
make branch_coverage
```
